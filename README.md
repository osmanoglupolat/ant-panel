## ğŸ“¦ Installation

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** 18.17.0 or higher (LTS version recommended)
- **pnpm** (recommended), npm, yarn, or bun package manager

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/ant-panel.git
cd ant-panel
```

### Step 2: Install Dependencies

Using pnpm (recommended):

```bash
pnpm install
```

Or using npm:

```bash
npm install
```

Or using yarn:

```bash
yarn install
```

Or using bun:

```bash
bun install
```

### Step 3: Environment Setup

Create a `.env.local` file in the root directory:

```env
# Application
NEXT_PUBLIC_APP_NAME=Ant Panel
NEXT_PUBLIC_APP_URL=http://localhost:3000

# Environment
NODE_ENV=development
```

> **Note:** Currently, no environment variables are required. The template works out of the box with mock data.

### Step 4: Run the Development Server

```bash
pnpm dev
```

Or using npm:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Step 5: Build for Production

```bash
pnpm build
pnpm start
```

## âš™ï¸ Configuration

### Theme Customization

The template includes 7 pre-built themes. To customize or add new themes:

1. Create a new CSS file in `app/theme/`
2. Update the theme selector in `components/layouts/theme-switcher/`
3. Add theme metadata in the theme configuration

### Environment Variables

Create a `.env.local` file for environment-specific configuration:

```env
NEXT_PUBLIC_APP_NAME=Ant Panel
NEXT_PUBLIC_APP_URL=http://localhost:3000
NODE_ENV=development
```

### Next.js Configuration

The `next.config.ts` file includes optimizations for:
- Image optimization (AVIF/WebP support)
- Compression
- Package imports optimization
- Bundle analysis
